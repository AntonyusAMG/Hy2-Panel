Подключение для клиента

Две ссылки: 1) HTTPS-подписка (лучше для HAPP — трафик и название) и 2) одна строка hysteria2:// для ручного импорта (QUIC). Для подписки в HAPP: удалите старую → «+» → вставьте HTTPS URL → обновить.

Должна начинаться с https:// того же хоста, что и панель. Если не открывается — проверьте public_base_url в config.json агента и прокси Apache (см. обзор: «Домен для ссылок HY2»).

Нет ключа подписки: задайте jwt_secret (длиной ≥ 8) в config.json агента — тогда появится HTTPS-ссылка.
